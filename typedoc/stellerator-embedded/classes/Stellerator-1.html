Class Stellerator

The stellerator class and namespace. In a typical application, a single instance is created and bound to a canvas element. This instance can than be used to run many different ROMs during its lifetime.

Notes on reading this documentation:

  • All code examples are ES6
  • The actual emulation runs on a web worker, and all methods that control emulation are asynchronous and return ES6 Promises
  • Check out the microevent.ts documentation for the event API

Basic example:

    const stellerator = new Stellerator(
        document.getElementById('stellerator-canvas'),
        'js/stellerator_worker.js'
    );

    stellerator.run(rom, Stellerator.TvMode.ntsc);

  • Creates an instance of Stellerator.

    Parameters

    • canvasElt: HTMLCanvasElement = null

      The canvas element that is used to display the TIA image. The height and width attributes of the canvas will be automatically maintained by Stellerator, so there is not requirement to set those to specific values.

    • workerUrl: string

      The URL from which the web worker will be loaded. In order to avoid cross domain issues, the worker should be hosted on the same domain as the stellerator build, and this parameter should read e.g. js/stellerator_worker.js

    • config: Partial<Config> = {}

      Optional configuration to customize emulator behavior. See [[Config]] for a full explanation of the values and their default.

frequencyUpdate

frequencyUpdate: EventInterface<number>

Subscribe to this event to receive periodic updates on the frequency of the emulated system. The unit is Hz. Check out the microevent.ts documentation on the event API.

Example (using JQuery to display emulation speed):

    stellerator.frequencyUpdate.addHandler(
        frequency => $('emulation-speed').text(`System speed: ${(frequency / 1e6).toFixed(2)} MHz`)
    );

stateChange

stateChange: EventInterface<State>

This event is dispatched whenever emulation state changes. Check out the microevent.ts documentation on the event API.

Example (using JQuery to display an error message):

    stellerator.stateChange.addHandler(
        state => {
            if (state === Stellerator.State.error) {
                 $('error-message').text(stellerator.lastError().message);
            }
        }
    );
